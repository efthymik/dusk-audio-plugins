# Multi-Q screenshots

Multi-Q has three modes (Digital, British, Tube) with very different layouts, so there is no single "main UI" shot. Each shot below is mode-specific or feature-specific.

| Filename | What it shows |
| --- | --- |
| `01-mode-digital.png` | Digital mode UI with the EQ curve and 8-band controls visible. Default state. |
| `02-mode-british.png` | British mode UI, showing the 4-band parametric layout with HPF/LPF and the Brown/Black switch. |
| `03-mode-tube.png` | Tube mode UI with LF Boost/Attenuation, HF Boost/Attenuation, Mid Dip/Peak, and Tube Drive controls. |
| `04-dynamic-eq-controls.png` | Digital mode with a band's right-click menu open, showing the Dynamic EQ controls (Threshold, Attack, Release, Range). Used in the dynamic de-essing workflow. |
| `05-tube-boost-cut.png` | Tube mode with both LF Boost and LF Attenuation engaged at the same frequency (60 Hz). The resulting EQ curve should be visible above the controls. Used in the "Vintage bass with the boost-and-cut trick" workflow. |
