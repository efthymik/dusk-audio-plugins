# Multi-Comp screenshots

Multi-Comp's UI changes per mode (8 modes plus Multiband), so there is no single "main UI" shot. Each shot below is mode-specific.

| Filename | What it shows |
| --- | --- |
| `01-mode-dropdown.png` | The Mode dropdown opened so all 8 mode names are visible (Vintage Opto, Vintage FET, Classic VCA, Vintage VCA (Bus), Studio FET, Studio VCA, Digital, Multiband). Used in Quick Start to establish that the plugin is multi-mode. |
| `02-mode-vintage-opto.png` | Vintage Opto mode with Peak Reduction at 50%, default Gain. Used in the "Lead vocal with Vintage Opto" workflow. |
| `03-mode-bus-comp.png` | Vintage VCA (Bus) mode with the "Classic Drum Glue" preset loaded (4:1 ratio, 30 ms attack, Auto release, 90 Hz sidechain HP). Used in the drum-bus workflow. |
| `04-mode-multiband-default.png` | Multiband mode with all 4 bands enabled, default 200 Hz / 2 kHz / 8 kHz crossovers. The four-band view. |
| `05-mode-multiband-collapsed.png` | Multiband mode with two bands disabled (e.g., Low and Hi-Mid), so the layout collapses to a 2-band view with the ON-strip toggles visible at the top. Used in the Multiband workflow and the issue #79 feature. |
| `06-presets-menu.png` | Preset menu opened, showing the 6 categories (Vocals, Drums, Bass, Guitars, Mix Bus, Creative) and a few preset names. |
